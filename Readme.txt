Esse programa foi desenvolvido em Linguagem C.

O programa carrega em memória os dados fornecidos no arquivo aps.txt e cria uma lista duplamente encadeada onde cada nó possui um campo chamado cpf que contém o valor carregado de uma linha do arquivo de dados e um vetor de ponteiros para os nós da lista.
Depois de carregados os dados, eles são ordenados no vetor por meio do algoritmo MERGE SORT. Após a ordenação, o programa solicita que o usuário digite um CPF. O dado lido do teclado será pesquisado e, caso encontrado, eliminado da lista encadeada.
Um novo CPF deverá ser solicitado para eliminação até que o usuário digite 0 (zero). Ao final, o programa informa quantos elementos restaram na lista encadeada.
